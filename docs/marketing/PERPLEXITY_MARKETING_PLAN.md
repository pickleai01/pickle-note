# 피클노트 마케팅 실행 계획 (한국어 시장 공략)

> 출처: Perplexity AI
> 작성일: 2026-02-22
> 목적: 한국어 시장 선점을 위한 구체적 마케팅 액션 플랜

**핵심 메시지**: "Glasp 하루 3개? 피클노트 무제한 YouTube → 학습노트 자동화"

---

## 타깃 오디언스 (우선순위순)

1. **수험생/대학생** — 수능/토익/자격증 강의
2. **직장인 자기계발** — 유데미/인프런 강의
3. **콘텐츠 크리에이터** — 자체 영상 분석
4. **강사/코치** — 강의 콘텐츠 정리

---

## 플랫폼별 마케팅 액션 플랜

### 1. Chrome Web Store (최우선 — 70% 트래픽 예상)

**스토어 페이지 최적화 (즉시 실행)**

| 항목 | 내용 |
|------|------|
| 제목 | "유튜브 무제한 학습노트 자동화 - NotebookLM 연동" |
| 아이콘 | 노트북+유튜브 로고 조합 (밝은 파랑) |
| 설명 첫 문단 | "Glasp 하루 3개 제한? 피클노트는 무제한! 수능 강의 → 시험노트 1클릭 완성" |

**스크린샷 5장 구성**:
1. Glasp 비교표 (하루 3개 vs 무제한)
2. 8가지 모드 데모 GIF
3. 실패 시 fallback 화면
4. 한국어 학습노트 결과물

**한국어 키워드**: "유튜브 노트북lm", "강의 자동 요약", "유튜브 학습노트", "Glasp 대체", "무제한 요약"

**목표**: 심사 통과 후 첫달 5,000 설치

---

### 2. 생산성 커뮤니티 (바이럴 효과)

**디시인사이드/에펨코리아/뽐뿌 (즉시 시작)**
- 제목: "[공짜] 유튜브 강의 자동 학습노트 만들어주는 크롬 확장"
- 본문: Glasp 비교 + 수능/토익 데모 GIF 첨부
- 해시태그: #유튜브 #공부 #크롬확장 #노트북lm

**네이버 카페 (지속)**
- 수만휘, 오르비, 자격증 카페
- 제목: "[공유] 인강 1클릭 학습노트 자동화 툴 발견"
- 3개 카페씩 매주 포스팅 → 총 50개 커뮤니티

**목표**: 첫주 1,000 설치 유입

---

### 3. 유튜버/블로거 제휴 (레버리지)

**유튜브 채널 협업 (1주 내 10곳 컨택)**
- 타깃: 생산성/공부법/자기계발 채널 (구독자 5K~50K)
- 제안: "무료 홍보 → 후원 링크 + 제휴 수익 30%"
- 샘플 스크립트: "여러분이 추천하던 Glasp 대체재! 하루 3개 제한 없이 무제한 학습노트"

**네이버 블로그 (지속적)**
- 공부법 블로거 20곳 스냅샷 후원 요청
- "수능 100일 남았다! 이 확장 하나면 강의 정리 끝"

**목표**: 3개월 내 10만 조회 → 1만 설치

---

### 4. 소셜 미디어 (바이럴 증폭)

**X (트위터) — 실시간 트렌드 공략**
- 해시태그: #공부 #수능 #자기계발 #생산성
- 스레드 형식: Glasp 문제점 → 피클노트 해결 → 데모 GIF → 설치 링크
- 매일 1개 강의 카테고리별 포스팅

**인스타그램 릴스 (시각적 데모)**
- 15초 GIF: "강의 클릭 → 학습노트 완성"
- 캡션: "하루 3개? 무제한으로 공부하자"

**목표**: 첫달 5만 노출 → 2,000 설치

---

### 5. SEO 콘텐츠 마케팅 (장기 트래픽)

**네이버 블로그/티스토리 (지속)**
- "2026 수능 완벽 정리법 3가지"
- "유튜브 인강 10배 효율화 크롬 확장 TOP3"
- 검색량 높은 키워드 공략

**목표**: 6개월 후 월 3,000 유기적 설치

## 주의사항

**하지 말 것**:
- "최고의 확장" 과대 광고 → 리뷰 1점 폭탄
- 스팸성 포스팅 → 커뮤니티 차단
- DOM 깨짐 방치 → "무료라서 그런가" 평판

**필수**:
- 매일 DOM 동작 테스트
- 사용자 리뷰 즉시 답변
- 실패율 5% 미만 유지
