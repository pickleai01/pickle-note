개인정보처리방침

시행일: 2026-02-20 | 최종 수정: 2026-02-20

1. 수집하는 개인정보

본 확장 프로그램은 어떠한 개인정보도 수집, 저장, 전송하지 않습니다.

  • 서버로 데이터를 전송하지 않습니다.
  • 사용자 계정 정보에 접근하지 않습니다.
  • 쿠키를 생성하거나 추적하지 않습니다.
  • 분석 도구(Google Analytics 등)를 사용하지 않습니다.

2. 로컬 데이터 저장

확장 프로그램은 아래 데이터를 사용자 브라우저 내 로컬에만 저장합니다:

  • chrome.storage.local: 노트북 URL, 노트북 목록, 사용자 설정(언어, 분량), 커스텀 프롬프트
  • chrome.storage.session: 임시 자동화 상태 (탭 닫으면 삭제)

이 데이터는 외부로 전송되지 않으며, 확장 프로그램 삭제 시 모두 제거됩니다.

3. 권한 사용 목적

  • tabs: 현재 탭 URL 확인 (YouTube 영상 감지)
  • scripting: NotebookLM 페이지에 자동화 스크립트 실행
  • activeTab: 활성 탭 정보 접근
  • storage: 로컬 설정 저장
  • notifications: 자동화 완료 알림 표시

모든 권한은 확장 프로그램 기능 수행에만 사용되며, 사용자 데이터 수집 목적으로 사용되지 않습니다.

4. 호스트 권한

  • *://*.youtube.com/*: YouTube 영상 URL 감지 및 제목 추출
  • *://notebooklm.google.com/*: NotebookLM 페이지 자동화 (소스 추가, 프롬프트 입력)

5. 제3자 데이터 공유

본 확장 프로그램은 제3자에게 어떠한 데이터도 공유, 판매, 전송하지 않습니다.

6. 데이터 보안

모든 데이터는 사용자 브라우저 내 Chrome Storage API를 통해 저장되며, Chrome의 보안 샌드박스 내에서만 접근 가능합니다.

7. 아동 개인정보

본 확장 프로그램은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 변경사항 고지

개인정보처리방침이 변경될 경우, 이 페이지를 통해 고지합니다. 중요한 변경 시 확장 프로그램 업데이트 노트에도 명시합니다.

Privacy Policy

Effective: 2026-02-20 | Last updated: 2026-02-20

1. Information We Collect

This extension does not collect, store, or transmit any personal information.

  • No data is sent to any external server.
  • No user account information is accessed.
  • No cookies are created or tracked.
  • No analytics tools (Google Analytics, etc.) are used.

2. Local Data Storage

The extension stores the following data locally in the user's browser only:

  • chrome.storage.local: Notebook URL, notebook list, user preferences (language, length), custom prompts
  • chrome.storage.session: Temporary automation state (cleared when tab closes)

This data is never transmitted externally and is removed when the extension is uninstalled.

3. Permission Usage

  • tabs: Read current tab URL to detect YouTube videos
  • scripting: Execute automation scripts on NotebookLM pages
  • activeTab: Access active tab information
  • storage: Store local preferences
  • notifications: Show automation completion notifications

All permissions are used solely for extension functionality. No permissions are used for data collection.

4. Host Permissions

  • *://*.youtube.com/*: Detect YouTube video URLs and extract titles
  • *://notebooklm.google.com/*: Automate NotebookLM (add sources, input prompts)

5. Third-Party Data Sharing

This extension does not share, sell, or transfer any data to third parties.

6. Data Security

All data is stored via Chrome Storage API within the user's browser, accessible only within Chrome's security sandbox.

7. Children's Privacy

This extension does not knowingly collect personal information from children under 13.

8. Changes to This Policy

Changes will be posted on this page. Significant changes will also be noted in extension update notes.

9. Contact

Privacy inquiries: privacy@피클AI.com
