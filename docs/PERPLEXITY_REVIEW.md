# 피클노트 (Pickle Note) v2.0.0 — 외부 AI 냉정 평가 리포트

> 출처: Perplexity AI
> 평가일: 2026-02-22
> 목적: 객관적/비판적 시각에서 제품 경쟁력 검증

---

## 1. 시장성 판단

**핵심 문제**: NotebookLM 니치 내 "YouTube → NotebookLM 자동화" 서브니치가 너무 좁음

- NotebookLM 2025년 1억 2천만 사용자 성장에도 불구하고 Notion/Google Docs급 메인스트림과 비교 시 여전히 틈새시장
- YouTube 요약 시장은 Glasp, Notta, TubeOnAI 등으로 과포화 → "NotebookLM에 넣어준다"는 우회 경로의 실질적 수요 불명확
- **천장**: NotebookLM 전체 사용자 중 YouTube+학습/연구 세그먼트 → 절대 규모 매우 제한적

---

## 2. 경쟁 환경

**직접 경쟁자**: Glasp(700K+ 사용자), Notta, TubeOnAI 등 YouTube summarizer들

| 경쟁자 특징 | 피클노트 | 차별점 의미 |
|-------------|----------|-------------|
| YouTube 내 사이드바 요약 | 없음 | 사용자 즉시성에서 패배 |
| 자체 LLM 요약+내보내기 | NotebookLM 의존 | 독립성 부족 |
| GPT/Claude 멀티모델 | 단일 NotebookLM | 선택권 제한 |

**즉사 리스크**: Google이 NotebookLM에 YouTube URL 자동 분석만 추가하면 핵심 가치 80% 증발

---

## 3. 기술적 리스크 (가장 치명적)

**DOM 자동화의 본질적 취약성**:

```
Google UI 변경 → selectors.js 수정 → 테스트 → 재배포 → 사용자 불만
↓ 반복
최종: 개발자 포기 → 서비스 방치
```

- Angular SPA controlled input 우회 → 내부 핸들러 변경 시 동작 불가
- Content Script 차단 정책 강화 시 전체 무력화
- **유지보수 현실**: "1파일 수정"이 아니라 "플로우 전체 재분석" 필요
